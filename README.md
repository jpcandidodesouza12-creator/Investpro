# 🪙 InvestPro - Gestão de Investimentos

O **InvestPro** é uma plataforma moderna de monitoramento de ativos financeiros. O sistema oferece uma visão consolidada de patrimônio, análise de alocação e cotações em tempo real através de uma interface intuitiva e de alta performance.

---

## 🏗️ Arquitetura (Monorepo)

Este projeto utiliza uma estrutura de **Monorepo**, facilitando a manutenção e o versionamento conjunto das camadas da aplicação:

* **`/frontend`**: Aplicação Single Page (SPA) construída com **React 18** e **Vite**. Focada em UX, utiliza gráficos interativos e design responsivo.
* **`/backend`**: API robusta responsável pela lógica de negócios, autenticação e integração com provedores de dados financeiros.

---

## ✨ Funcionalidades Principais

* **Dashboard Inteligente**: Visualização de KPIs (Patrimônio Total, Rendimento, Lucro/Prejuízo).
* **Gráficos de Alocação**: Distribuição visual da carteira por classe de ativos.
* **Gestão de Carteira**: Adição, remoção e edição de ativos em tempo real.
* **Cotações Live**: Integração com APIs financeiras para atualização de preços.
* **Autenticação Segura**: Fluxo de login completo com proteção de rotas.

---
